This is a web project focusing on generative art and human-computer interaction related web works, i.e., presenting my personal visual and interactive ideas through the vehicle of web pages.

I was inspired by Antfu[1]'s 100 days[2] project to do this project. I've been doing some web work on and off since I started working in the front-end space, some of which has been published on XueFeng on CodePen[3] and some of which has been in various GitHub projects, so it's particularly distracting. When I saw a project like 100days, I thought it was time to centralize this kind of work and manage it, in addition to the work I've already written, there's still a lot of prototypes lying around in the idea list.
